Gestor de Vehículos en C#

Aplicación de consola desarrollada en C# que permite gestionar una lista de vehículos
incluyendo funciones como añadir, mostrar y eliminar vehículos.


Requisitos

Visual Studio 2022 o superior
.NET SDK instalado (versión 6.0 o superior recomendada)

Estructura del proyecto:

GestorVehiculos/
├── Program.cs
├── Vehiculo.cs
├── GestorVehiculos.cs
├── GestorVehiculos.csproj
└── README.md

Funcionalidades

Registrar, mostrar, eliminar, salir.
Coche, moto, camión.
Marca, modelo, año, precio, Nº puertas. 
Mostrar lista de coches, motos o camiones
Eliminar coche, moto o camión a través de su número de referencia
Validaciones básicas en consola


Cómo ejecutar el programa:

Opción 1: Desde Visual Studio

Clona o descarga el repositorio.
Abre el archivo GestorVehiculos.csproj 
Compila el proyecto (Ctrl + Shift + B).
Ejecuta (F5 o botón "Iniciar").


Notas
Este proyecto no incluye el ejecutable (.exe) para mantener el repositorio limpio. Si deseas probarlo, sigue las instrucciones anteriores para compilarlo tú mismo.
